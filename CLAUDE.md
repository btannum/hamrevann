# Hamrevann

Landingsside for boligprosjektet Hamrevann – 16 tomannsboliger i Kristiansand.
Bygget av Reach Media for utbygger Future Green Group / Konsmo Fabrikker.
Megler: Eiendomsmegler Norge (Tor Even Kristensen og Audun Remesvik).

## Teknologi

- Ren HTML/CSS/JS – én enkelt fil (`index.html`), ingen build-steg eller avhengigheter

## Kjøre lokalt

```bash
cd ~/Prosjekter/hamrevann
python3 -m http.server 8080
```

Åpne http://localhost:8080 i nettleseren.

## GitHub

Repository: https://github.com/btannum/hamrevann

## Struktur

- Hero med bakgrunnsbilde og CTA-knapper
- Statusbar med byggestatus
- Fordeler-seksjon (3 kort)
- Bildegalleri med lightbox (6 bilder fra filesafe.space CDN)
- Beliggenhet med avstandsliste
- Spesifikasjoner (16 enheter, 94–112 m², fra 5,5 mill.)
- Kontaktskjema via HighLevel (LeadConnector iframe)
- Samarbeidspartnere og meglerkort
- Sticky CTA på mobil
